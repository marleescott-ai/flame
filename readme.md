# FLAME – Beginning HTML, CSS and JavaScript

Welcome to **FLAME** — the **Fire League of Arcane Mages & Engineers**.

FLAME is a beginner-friendly web development course that teaches students how to build websites and web applications using **HTML**, **CSS**, and **JavaScript**. Rather than learning through disconnected exercises, students work within a consistent fantasy-political world, gradually developing the skills needed to create modern interactive websites.

The course has been designed for secondary school learners and introduces web development through a series of practical tasks that build on one another. By the end of the course, students will have created multi-page websites, styled layouts, interactive forms, and JavaScript-powered applications.

---

# Learning Objectives

Students will learn to:

* Create webpages using HTML
* Structure content using semantic markup
* Apply CSS styling using selectors, classes, and IDs
* Build multi-page websites
* Create reusable design components
* Use Flexbox and Grid layouts
* Capture user input
* Display output dynamically
* Store information in variables
* Use selection (`if` statements)
* Use iteration (`for` and `while` loops)
* Store data in arrays
* Debug existing code
* Develop a complete web application project

---

# Teaching Approach

FLAME uses several evidence-informed teaching approaches:

* PRIMM (Predict, Run, Investigate, Modify, Make)
* Guided examples
* Incremental skill development
* Consistent thematic context
* Worked examples
* Debugging activities
* Open-ended extension opportunities

Each task focuses on a small number of new concepts while reinforcing previously learned skills.

# Repository Structure

Each task is contained within its own folder.

Typical task folders contain:

```text

```

The `task.txt` file contains student instructions.

Tasks are intended to be completed sequentially.

# Course Overview

## 01 – The Flame Guild

**Focus:** Basic HTML

Students create their first webpage and learn:

* Headings
* Paragraphs
* Lists
* Images
* Hyperlinks

This task introduces the FLAME setting and the structure of a simple webpage.

## 02 – Briana the Embermage

**Focus:** Semantic HTML

Students create a character profile page while learning:

* Strong and emphasis tags
* Quotations
* Ordered and unordered lists
* Images
* Content structure

The emphasis is on choosing appropriate HTML elements.

## 03 – Maelor the Spellwright

**Focus:** Introduction to CSS

Students begin styling webpages using:

* CSS selectors
* Classes
* Colours
* Typography
* Spacing

This task introduces the separation of content and presentation.

## 04 – Navbar Styling

**Focus:** Multi-page websites

Students create navigation systems and learn:

* Relative links
* Shared stylesheets
* Navigation menus
* Consistency across pages

Files include:

```text
index.html
products.html
services.html
shop.html
```

Students experience how larger websites are organised.

## 05 – The Flame Campaign

**Focus:** Design decomposition and layout

Students recreate a professional website design from supplied mock-ups.

Topics include:

* Design analysis
* Wrappers and containers
* Reusable cards
* CSS Grid
* Flexbox
* Responsive layout thinking

This task represents the transition from simple pages to real website design.

---

# JavaScript Section

The next tasks transition from static sites into programming.

## 06 – Join the Party

**Focus:** JavaScript inputs and outputs**

Students create their first interactive webpage.

Topics include:

* Variables
* User input
* Reading form values
* Updating page content

Students learn how JavaScript can change a webpage dynamically.

## 07 – Party Sign Up Form

**Focus:** Working with multiple inputs

Students collect information from users using forms.

Topics include:

* Text inputs
* Combining values
* Displaying personalised output

This task reinforces the input–process–output model.

## 08 – Contribute to the Campaign

**Focus:** Processing data

Students perform calculations using user-entered values.

Topics include:

* Numbers
* Arithmetic operators
* Variables
* Displaying calculated results

Students begin seeing programs as systems that transform data.

## 09 – Can You Vote?

**Focus:** Selection

Students learn how programs make decisions.

Topics include:

* `if` statements
* `else`
* Comparison operators
* Boolean logic

Students determine whether users are eligible to vote based on supplied information.

## 10 – Find Your Party Role

**Focus:** More complex decision making

Students create a role-matching application.

Topics include:

* Multiple conditions
* Nested decisions
* User classification

This task extends students’ understanding of selection.

## 11 – Party Policies

**Focus:** Arrays

Students learn to work with collections of data.

Topics include:

* Arrays
* Indexes
* Accessing stored values
* Displaying data from arrays

Students move beyond working with single pieces of information.

## 12 – Flamejack

**Focus:** Iteration

Students learn how programs repeat actions.

Topics include:

* `for` loops
* `while` loops
* Repetition
* Generating multiple outputs

Iteration is one of the most important concepts in programming and allows students to process collections of data efficiently.

## 13 – Final Task

**Focus:** Bringing everything together

Students create a complete web application that demonstrates their understanding of:

* HTML
* CSS
* Variables
* Input and output
* Selection
* Arrays
* Iteration

This task serves as the culmination of the course.

# Setup

Before starting, install the following software:

* **Visual Studio Code**
  https://code.visualstudio.com/

* **Git**
  https://git-scm.com/downloads

The default installation options are fine.


## Fork your own version of the repo

## Clone the Repository

Open a terminal and run:

```bash
mkdir dev
cd dev
git clone https://github.com/[your username]/flame.git
cd flame
code .
```

This will:

* Create a development folder
* Download the FLAME course files
* Open the project in VS Code

## Install Live Server

In VS Code:

1. Open the **Extensions** panel
2. Search for **Live Server**
3. Install the extension by **Ritwick Dey**

To preview a webpage:

1. Open an `index.html` file
2. Right-click the file
3. Select **Open with Live Server**

Your browser will automatically refresh whenever you save changes.

## Getting Started

Complete the folders in order:

```text
01_the_flame_guild
02_briana_the_embermage
03_maelor_the_spellwright
...
13_final_task
```

Each folder contains the files and instructions needed for that task.
